# Example request

Build a Chrome extension with these requirements:

- When the user clicks the toolbar icon, read the current page title, URL, and the first 3 content paragraphs
- Show the result in the popup
- Allow saving the result to local storage
- Only work on the current active page when explicitly triggered by the user
- No backend required
- Prefer plain JavaScript over React
- Provide the manifest, project structure, core code, and debugging steps
